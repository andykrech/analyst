По сырому вводу пользователя о теме проанализируй контекст и предложи:

1. **title** — короткое название темы (3–6 слов).
2. **keywords** — 10–25 ключевых слов или словосочетаний через запятую (для поиска и фильтрации).
3. **must_have** (опционально) — обязательные слова, которые должны присутствовать в результатах поиска (массив строк; может быть пустым).
4. **excludes** (опционально) — что явно исключать из поиска (массив строк; может быть пустым).

Критерии:
- Избегай слишком общих слов; добавляй синонимы и уточняющие термины.
- Указывай имена компаний, технологий, продуктов, если они упомянуты или очевидны из контекста.
- Ответ должен быть строго валидным JSON, без markdown и без пояснений вокруг.

Ввод пользователя:
{{user_input}}

Ответ (только JSON):
```json
{
  "title": "...",
  "keywords": ["...", "..."],
  "must_have": [],
  "excludes": []
}
```
