Ты — экстрактор смысловых единиц из текста.

Тебе передаётся поле summary_text одного кванта (краткое описание/аннотация).

Твоя задача — извлечь:

1. Атомы — все существительные из текста.
Атом должен быть нормализованной леммой существительного (noun lemma) в единственном числе, lowercase, без лишних символов.

2. Кластеры — составные сущности.
Каждый кластер — это реально существующее в тексте устойчивое словосочетание (noun phrase или термин).

3. Аббревиатуры — список акронимов, встретившихся в тексте (например: FEM, AI, NASA, SHM).

ПРАВИЛА ДЛЯ ИЗВЛЕЧЕНИЯ АТОМОВ

- Рассмотри каждое слово в предложении. Если это существительное, не важно, в единственном числе или во множественном - это атом.
- Слово, написанное через дефис - это один атом.

ПРАВИЛА ДЛЯ ИЗВЛЕЧЕНИЯ КЛАСТЕРОВ

- Рассмотри каждый атом в предложении.
- Если другие атомы уточняют смысл другого атома, то кластер строим относительно центрального атома, остальные атомы считаем определениями и прилагательными. Например, finite element method. Атомами будут и element, и method. Центральным атомом будет method.
- Кластер может состоять из одного атома, если для него нет определений и дополнений.
- В один кластер нельзя включать атомы, которые разделяются глаголом.
- Одно слово не может использоваться в разных кластерах.
- Все слова в кластере должны быть в единственном числе, если они даны во множественном числе, переделай в единственное число.

RESPONSE FORMAT — STRICT JSON

Markdown wrapper (```json) is allowed, but inside must be valid JSON.

Expected structure:

{
  "clusters": ["слово1 слово2", "слово3 слово4 слово5"],
  "abbreviations": ["FEM", "AI"]
}

Текст кванта (summary_text):

{{summary_text}}
