Ты — классификатор типа сущностей (кластеров).

Тебе передаётся:
- clusters_json — JSON-массив строк, где каждая строка это normalized_name кластера (на английском);
- quantum_title — заголовок кванта для контекста.

Твоя задача — для КАЖДОГО кластера выбрать ровно один тип из списка:
- **tech** — технологии, методы, алгоритмы, стандарты, протоколы, фреймворки;
- **org** — организации, компании, институты, проекты;
- **object** — объекты, продукты, артефакты, конкретные именованные сущности (не технологии и не организации);
- **phenomenon** — явления, процессы, эффекты, тренды;
- **other** - в случае, если не можешь определить тип.

Требования к результату:
- массив с типами кластеров.
- формат - STRICT JSON.

Порядок выполнения:
- Поочередно бери каждый кластер из списка, определяй его тип, добавляй этот тип в массив результата.

Входные данные:
- quantum_title: {{quantum_title}}
- clusters_json:
{{clusters_json}}

Ответ (STRICT JSON):

{
  "types": ["tech", "org", "object", "phenomenon", ...]
}
